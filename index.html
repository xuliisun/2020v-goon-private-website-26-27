// 01 — About

Built to Compete.

Team 2020V is a competitive VEX Robotics team driven by engineering precision and a relentless pursuit of performance. Every mechanism, every line of auton code, every driver skill run is the result of hundreds of hours of iteration.

We don't just build robots — we solve problems under pressure. Our team combines mechanical design, electronics, and software to field a machine that's fast, reliable, and built to win.

From our first competition scrimmage to qualification events and regionals, 2020V competes at every level with the same intensity.

// 02 — The Machine

This Season's Robot.

DRIVETRAIN
6-Motor Tank Drive

High-torque 6-motor drivetrain with 4" omni wheels optimized for pushing power and field traversal speed. Locked in for driver control consistency.

INTAKE
Flex Wheel Intake

Rubber flex wheels run at maximum RPM for aggressive game object collection. Adjusted spacing ensures consistent indexing with zero jams under match pressure.

SCORING
Catapult Launch System

Slip-gear catapult tuned for match load shots with a repeatable flight arc. Minimal charge time between shots keeps cycle counts high.

AUTONOMOUS
PID + Odometry

Closed-loop PID control with odometry tracking for precise positioning. Multiple auton routines selectable pre-match based on alliance strategy.

PNEUMATICS
Endgame Deployment

Pneumatic-actuated endgame mechanism fires in under 0.3 seconds. Pressure tuned for reliability across repeated activations during skills runs.

ELECTRONICS
Brain + V5 Smart Motors

Full V5 ecosystem with real-time telemetry monitoring. Motor thermals tracked and rotation sensors on all critical subsystems for data-driven tuning.
